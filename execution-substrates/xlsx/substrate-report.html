What This Substrate Does

The XLSX substrate generates native Excel workbooks with embedded formulas, allowing the rulebook logic to execute directly within Microsoft Excel.

Orchestration Pattern

  1. Generate: inject-into-xlsx.py creates Excel workbook using openpyxl
  2. Inject Formulas: Converts rulebook formulas to Excel cell references
  3. Test: Opens workbook and validates calculated values match expected

Key Features

  • Native Excel Formulas: Calculations run directly in Excel
  • Cell Reference Translation: {{FieldRef}} becomes $B2, etc.
  • Multi-Sheet Support: Each entity gets its own worksheet
  • Portable: Works offline without any dependencies

Formula Translation

Rulebook formulas are converted to Excel formulas with cell references:

Rulebook: =SUBSTITUTE(LOWER({{DisplayName}}), " ", "-")
Excel: =SUBSTITUTE(LOWER($B2), " ", "-")
Rulebook: ={{FieldA}} & " - " & {{FieldB}}
Excel: =$C2 & " - " & $D2
